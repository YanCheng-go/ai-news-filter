Welcome to MyFocalAI

Your personal AI news intelligence. Aggregates content from Twitter, YouTube, arXiv, RSS & more, scored by relevance to your interests (scoring supported locally for now).

Get Started

1

Create an account

Click the icon in the top-right corner to sign up with your email. Your feed is private and personalized.

2

Add your sources

Once signed in, go to Manage Sources from the user menu. Add Twitter accounts, YouTube channels, RSS feeds, arXiv queries, or load our curated defaults.

3

Fetch & score

Hit Fetch Now to pull the latest content. Each item is scored 0–100 against your interests by an LLM. Scoring is supported in local mode for now.

4

Browse & filter

Use the filter buttons and search bar to find what matters. Red badges show new items since your last visit. Explore Trends, Events, and more from the nav.

What's on each page

How it works

The pipeline fetches content from Twitter/X, YouTube, arXiv, RSS feeds (via RSSHub), GitHub trending repos, and more. Each item is scored 0-100 against your priorities defined in config/principles.yml. Two modes: local (SQLite + Ollama + FastAPI) and cloud (Supabase + Vercel).

Tech stack

  • BackendPython, FastAPI, APScheduler
  • DatabaseSQLite (WAL mode) / Supabase Postgres
  • ScoringOllama (local) / Claude API (cloud)
  • IngestionRSSHub, aiohttp, BeautifulSoup
  • FrontendJinja2 templates (local), static HTML + JS (Vercel)
  • CI/CDGitHub Actions, Vercel
  • Dev toolsNix + direnv, uv, ruff
